# F5 Networking - Homepage

Modern and professional homepage for F5 Networking, a technology company specializing in:
- BPO Services (Call Center)
- AI Virtual Assistants
- ECHO - Omnichannel CRM
- SIP Telephony (PBX)

## Tech Stack
- Vite
- React
- Framer Motion
- Lucide React
- CSS Modules / Vanilla CSS

## Features
- Interactive digital particle background.
- "System Sync" explosion timer (20s).
- Professional dark aesthetic with glassmorphism.
